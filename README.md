<h1> Generador de Constancias de Eventos | Corporativo RIEC </h1>

Este repositorio corresponde al Generador de Constancias de Eventos de Youtube de la empresa Corporativo RIEC.

En este archivo se genera lo siguiente:
• Maquetación y estructura de interfaces web
• Uso de HTML, CSS y JavaScript
• Buenas prácticas de organización de código

El objetivo principal de este proyecto es que el Alumno genere su propia constancia y la reciba de manera rapida y oportuna, de esa manera un proceso de la empresa queda automatizado de manera parcial.
